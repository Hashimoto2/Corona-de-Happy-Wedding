# Corona de Happy Wedding

## サイト概要
ウエディング情報サイト

### サイトテーマ
コロナ禍に挙式を挙げられないお二人のお役立ちウエディングサイト

### テーマを選んだ理由
コロナ禍の為、通常に結婚式が挙げられないお二人に少しでも後悔なく素敵な結婚式を挙げてほしいと思った為

### ターゲットユーザ
これから結婚を考える男女、友人の結婚式に出席など結婚式に関わる人々

### 主な利用シーン
結婚式について情報が欲しいときや困った時や参考にしたい時など
